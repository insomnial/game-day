# Game Day
This messaging system for Slack collects the posted games of a URL and sends them formatted and with the correct timezone of the logged-in user to a specified channel in a Slack server.

## Women's NCAA Basketball
### Top-25 Games
Collects the current day's slate of NCAA women's basketball games for Division 1 Top 25 teams as posted by the NCAA's listings at https://www.ncaa.com/scoreboard/basketball-women/d1/2024/12/08/top-25.

### Rankings
Posts the weekly updated rankings on Mondays.

### March Madness 2025
Now supports pulling games during March Madness with global switches.
Now supports sending to debug channels with one global switch.

## 2026 Olympics
### Upcoming Events
Posts the upcoming day's events.
